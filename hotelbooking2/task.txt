
необходимо разработать web-приложение, позволяющее получать информацию о доступных к бронированию номерах отеля.

Приложение должно быть построено с использованием REST принципов, по запросу пользователя возвращается список номеров
удовлетворяющих критериям поиска. Список номеров приложение возвращает в формате JSON.

	в качестве инструмента сборки проекта использовать Maven. Приложение должно поддерживать многомодульную структуру
	код приложения должен быть покрыт Unit тестами (DBUnit, Mockito). Используйте TDD подход
	конфигурацию зависимостей выполнить с использованием Spring IoC

	в качестве СУБД использовать Oracle 11g. Для доступа к данным использовать Hibernate Framework (xml mapping)
	в качестве среды разработки использовать IDE Eclipse с подключенным SonarLint плагином
	приложение размещать на Apache Tomcat 9 версии

	необходимо подготовить схему модели данных (физический уровень)
	необходимо подготовить скрипт генерации структуры вашей БД
	необходимо подготовить скрипт заполнения вашей БД начальными данным

Разрабатываемое приложение должно размещаться в вашем git репозитории в отдельной ветке. Результат работы над приложением должен появляться в репозитории не реже одного раза в рабочий день.
